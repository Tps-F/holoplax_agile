# 価値のすり合わせ（たたき台）

## ここまでの合意
- 主対象: 個人のセルフマネジメント
- ワークスペース: コンテクスト分離のために利用
- 価値: 認知負荷の軽減 / 継続の支援 / 成果の最大化
- 差別化: AIの先回り下準備 + アジャイルの運用定着 + 障害除去
- 中心ループ: Capture / Plan（実行は人間、AIは準備と計画支援）
- 指標: タスク進行量 / Sprint Velocity / PBI Velocity

## まだ決めたいこと
- 「成果の最大化」をどう測るか（速度/達成率/継続率のどれに寄せるか）

## 北極星指標（合意）
- ベロシティの最大化とその維持（長期の継続性を重視）

## 北極星指標の候補例
- PBI消化率 × Sprint Velocity（計画と実行の両方を反映）
- 週次で「やるべきこと（3件）」が完了した割合
- 未完了数の減少率（認知負荷の減り方を測る）
